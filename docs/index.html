Parameter Tuning Examples

Tuning with 8 parameters

There are 2 engines involved, the test_engine and the base_engine. In every iteration an engine matches will be conducted. The test_engine will take the suggested parameter values from the tuner and use it to play against the base_engine. The base_engine will use the parameter values used by test_engine from the previous iteration.

Two matches will be conducted in every iteration to approximate the gradient and there are 8 games per match for a total of 16 games. It is better to have more games per match so that the tuner can give a good approximation of the parameter values to try.

Parameters to be tuned

Units are in centipawn, at the start queens are undervalued, rooks are overvalued, bishops are undervalued and finally knights are overvalued.

  • QueenValueOp 850
  • QueenValueEn 850
  • RookValueOp 600
  • RookValueEn 600
  • BishopValueOp 250
  • BishopValueEn 250
  • KnightValueOp 400
  • KnightValueEn 400
